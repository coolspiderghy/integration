Integration and Segregation

Supporting MATLAB Code

Download this project as a .zip file Download this project as a tar.gz file

This repo contains the code used to generate the results for the manuscript: 'Spatiotemporal Dynamics of Integration and Segregation in the Human Functional Connectome' by Shine, J.M., Bell, P.T., Koyejo, O., Gorgolewski, K.J., Moodie, C.A. and Poldrack, R.A.

The manuscript is currently undergoing peer review, however a version is available at: http://arxiv.org/abs/1511.02976.

For further questions, please feel free to contact us via email: macshine@stanford.edu.
